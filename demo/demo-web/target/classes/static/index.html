🚀 Atlas Demo Web API

这是一个基于Spring Boot的Web服务，提供HTTP接口来调用后端的gRPC微服务。

服务状态： 检查中...

📋 API 接口

GET /api/
获取API信息和可用端点
GET /api/health
健康检查，包含后端gRPC服务状态

👤 用户服务 (UserService)

GET /api/users/{userId}
获取用户信息
curl http://localhost:8080/api/users/user1
POST /api/users
创建新用户
curl -X POST http://localhost:8080/api/users \
-H "Content-Type: application/json" \
-d '{"name":"张三","email":"zhangsan@example.com"}'

📦 订单服务 (OrderService)

GET /api/orders/{orderId}
获取订单信息
curl http://localhost:8080/api/orders/order123
POST /api/orders
创建新订单
curl -X POST http://localhost:8080/api/orders \
-H "Content-Type: application/json" \
-d '{"userId":"user1","productName":"iPhone 15","amount":999.99,"quantity":1}'
GET /api/users/{userId}/orders
获取用户的所有订单
curl http://localhost:8080/api/users/user1/orders

🏗️ 架构说明

本Web服务作为API网关，将HTTP请求转换为gRPC调用：

  • atlas-demo-web:8080 - Web API服务 (本服务)
  • atlas-demo-up-server:9091 - UserService gRPC服务
  • atlas-demo-down-server:9092 - OrderService gRPC服务

🔍 链路追踪

所有API调用都包含分布式链路追踪功能，可以在日志中看到完整的调用链路。

🚀 快速开始

  1. 启动后端gRPC服务：
    cd atlas-demo-up-server && mvn spring-boot:run &
    cd atlas-demo-down-server && mvn spring-boot:run &
  2. 启动Web服务：
    cd atlas-demo-web && mvn spring-boot:run
  3. 访问 http://localhost:8080
